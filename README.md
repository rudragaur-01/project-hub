# Project Management System

A full-stack task and project management system where users can:

- Create projects
- Create boards and tasks
- Assign tasks to users
- Add comments
- Track task status (Todo → In Progress → Done)
- Collaborate within teams

Built using the MERN stack with REST APIs and role-based access.

---

## Tech Stack

### Frontend

- React
- Tailwind CSS / Chakra UI
- Axios
- React Router

### Backend

- Node.js
- Express
- MongoDB
- JWT Authentication
- REST APIs

---

## Features

✔ User authentication (JWT)  
✔ Role-based project access (Owner & Members)  
✔ Project creation and management  
✔ Task boards and task management  
✔ Task assignment and comments  
✔ Activity tracking  
✔ Secure REST architecture
